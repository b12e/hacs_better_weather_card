# Better Weather Card

A better weather card for Home Assistant with Mushroom-inspired design.

## Features

- **Mushroom-style design** - Clean and modern interface matching the Mushroom card aesthetic
- **Current weather display** - Shows temperature, humidity, and current conditions
- **Visual editor** - Easy configuration through the Lovelace UI

## Installation

### HACS (Recommended)

1. Make sure [HACS](https://hacs.xyz/) is installed
2. Add this repository as a custom repository in HACS:
   - Go to HACS â†’ Frontend
   - Click the three dots in the top right
   - Select "Custom repositories"
   - Add this repository URL
   - Select "Lovelace" as the category
3. Click "Explore & Download Repositories"
4. Search for "Better Weather Card"
5. Click "Download"
6. Refresh the Home Assistant dashboard


## Configuration

### Visual Editor (Recommended)

The card is **fully configurable from the Home Assistant UI** - no YAML required!

**To add and configure the card:**

1. Edit your dashboard
2. Click "Add Card"
3. Search for "Better Weather Card"
4. Click on it to add it to your dashboard
5. Configure the card by selecting a weather entity
6. Click "Save" and you're done!
